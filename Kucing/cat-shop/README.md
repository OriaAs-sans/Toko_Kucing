# Cat Shop (Laravel skeleton)

Ini skeleton awal untuk aplikasi sales toko kucing.
Saya membuat file scaffold. Untuk menyelesaikan instalasi Laravel dan menjalankan aplikasi, jalankan script PowerShell `setup.ps1` di bawah.

Quick setup (PowerShell):

```powershell
cd F:\VSCODE\Kucing\cat-shop
./setup.ps1
```

Isi `setup.ps1` akan menjalankan `composer create-project` untuk membuat Laravel, meng-copy file scaffold ke dalam project, dan menjalankan migrasi. Jika Anda tidak punya Composer/Node, install terlebih dahulu.

Lihat `setup.ps1` untuk langkah lengkap.
